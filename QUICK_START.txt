  • Code blocks → LTR by default
  • Chat inputs → RTL by default
  • Global page toggle
  • Perfect for Hebrew & Arabic speakers

💡 TESTING BEFORE SUBMISSION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Extract ZIP to a folder
2. Open chrome://extensions/
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select extracted folder
6. Go to claude.ai
7. Click extension icon
8. Verify buttons appear:
   ✓ Code blocks = LTR button (left side)
   ✓ Chat input = RTL button (left side)
   ✓ Copy button stays on right
9. Test toggling directions
10. Check console for errors (F12)
